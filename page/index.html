External Libraries


External tools and libraries developed outside of the Forum but used extensively by the Quant team.



echarts4r

The echarts4r package is an open source R package available on CRAN and that provides powerful interactive visualisations in R. It is essentially a wrapper around the echarts javascript visualisation library.

Documentation
timetk

Timetk is an amazing package that sonsolidates and extends time series functionality of the modeltime ecosystem for time series analysis and forecasting. It's made for easy visualization, wrangling, and feature engineering of time series data but also for forecasting and machine learning predictions.

Documentation
Brms

Brms package provides a STAN wrapper for R, which allows for posterior probability estimation using a Gibbs sampler with a Markov chain Monte Carlo (MCMC) algorithm for obtaining a sequence of observations which are approximated from a specified multivariate probability distribution, when direct sampling is difficult.

Documentation
